## 软件简介

PSO数据分析工具是一款专门用于**胫神经电刺激**实验数据分析和参数辨识的桌面应用程序。

### 核心功能

- **数据导入**：支持CSV/Excel格式的EMG数据
- **PSO优化**：自适应粒子群优化算法进行参数辨识
- **模型评估**：RMSE、MAE、R²等多维度指标
- **阈值计算**：计算达到目标响应概率所需的刺激电流
- **SD曲线**：刺激-剂量曲线可视化
- **多图表分析**：7种专业图表

### 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11, macOS 10.14+ |
| 内存 | ≥ 4GB RAM |
| 硬盘 | ≥ 500MB 可用空间 |
| Python | 3.9+ (已打包版本无需安装) |

## 快速开始

### 步骤1：安装与启动

```bash
# 运行程序（Python环境）
python main.py

# 或双击 run.bat (Windows)
```

### 步骤2：导入数据

1. 点击工具栏的 **"导入"** 按钮
2. 选择EMG数据文件（`emg_processed.csv`）
3. 系统自动解析并显示数据预览

### 步骤3：设置参数

在左侧面板调整PSO参数：

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 粒子数 | 50-100 | 粒子群规模，越大搜索能力越强 |
| 迭代次数 | 50-100 | 最大迭代次数 |
| 目标RMSE | 0.03 | 收敛阈值，越小精度要求越高 |
| 分析频率 | 10/20 Hz | 刺激频率 |

### 步骤4：运行优化

1. 点击 **"运行PSO优化"** 按钮
2. 观察进度条和状态栏
3. 优化完成后自动显示结果

### 步骤5：查看结果

- 在 **优化结果Tab** 查看参数和指标
- 在 **图表Tab** 查看各种可视化图表
- 使用工具栏按钮保存图表

## 功能详解

### 数据导入

#### 单文件模式
选择包含所有数据的CSV文件。

#### 双文件模式
同时导入：
- `stim_params.csv` - 刺激参数（脉宽、频率等）
- `emg_processed.csv` - EMG响应数据

#### 数据验证
系统会自动检查：
- 列名是否包含必需字段
- 数据是否为数值类型
- 数据点数量是否充足

### PSO优化

#### 算法原理
粒子群优化(Particle Swarm Optimization)是一种全局优化算法，通过模拟鸟群觅食行为寻找最优解。

#### 参数说明

| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 粒子数 | 10-200 | 50 | 搜索粒子数量 |
| 迭代次数 | 10-500 | 50 | 最大迭代代数 |
| 惯性权重w | 0.4-0.9 | 自适应 | 影响全局/局部搜索 |
| 学习因子c1 | 0.5-3.0 | 1.8 | 个体学习能力 |
| 学习因子c2 | 0.5-3.0 | 1.5 | 群体学习能力 |

#### 输出指标

| 指标 | 说明 | 理想值 |
|------|------|--------|
| RMSE | 均方根误差 | 越小越好 |
| R² | 决定系数 | 接近1 |
| MAE | 平均绝对误差 | 越小越好 |

### 阈值计算

根据辨识的参数模型，计算达到特定响应概率所需的刺激电流：

| 目标概率 | 典型应用 |
|----------|----------|
| P=50% | 感觉阈值 |
| P=80% | 运动阈值 |
| P=90% | 有效刺激 |

### SD曲线

刺激-剂量(Strength-Duration)曲线描述：
- 横轴：脉宽 (μs)
- 纵轴：阈值电流 (mA)

曲线特征：
- 连续刺激时呈双曲线形态
- 可用于确定**利用时值**(Chronaxie)和**基强度**(Rheobase)

## 数据格式

### EMG数据文件 (emg_processed.csv)

```csv
I_peak(mA),Freq_10Hz,Freq_20Hz
5.0,0.05,0.08
6.0,0.12,0.18
7.0,0.25,0.35
8.0,0.42,0.55
9.0,0.58,0.72
10.0,0.71,0.85
```

| 列名 | 类型 | 说明 | 示例 |
|------|------|------|------|
| I_peak(mA) | float | 峰值电流 | 5.0, 6.0... |
| Freq_10Hz | float | 10Hz响应概率 | 0.05-1.0 |
| Freq_20Hz | float | 20Hz响应概率 | 0.08-1.0 |

### 刺激参数文件 (stim_params.csv)

```csv
Parameter,Value,Unit
PW,200,us
Freqs,"10, 20",Hz
```

或简写格式：

```csv
PW,200
Freqs,"10, 20"
```

## 常见问题

### Q1: 导入数据时提示"数据格式错误"

**原因**：CSV文件列名不符合要求

**解决方法**：
1. 确保第一行是列名
2. 确保包含 `I_peak(mA)` 列
3. 确保有至少一个频率响应列（如 `Freq_10Hz`）

### Q2: 优化长时间不收敛

**原因**：
- 目标RMSE设置过低
- 粒子数不足
- 数据噪声过大

**解决方法**：
1. 适当提高目标RMSE
2. 增加粒子数和迭代次数
3. 检查数据质量

### Q3: 图表显示为空白

**原因**：
- 尚未运行优化
- 数据导入不完整

**解决方法**：
1. 先导入数据
2. 运行PSO优化
3. 图表将自动更新

### Q4: R²为负值

**原因**：模型拟合比简单均值更差

**解决方法**：
- 增加迭代次数
- 调整PSO参数
- 检查数据是否符合模型假设

### Q5: 如何保存结果？

**方法一**：菜单导出
- 文件 → 导出 → Excel/CSV/JSON

**方法二**：单独保存图表
- 在图表Tab点击"保存"按钮
