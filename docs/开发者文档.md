# 开发者文档

本文档为开发者提供PSO_UI项目的架构说明和开发指南。

---

## 项目架构

```
PSO_UI/
├── main.py                      # 主程序入口
├── ui/                          # UI层
│   ├── main_window.py          # 主窗口
│   ├── chart_widgets.py        # 图表组件
│   └── analysis_workflow.py    # 分析工作流
├── modules/                     # 业务逻辑层
│   ├── config.py               # 配置
│   ├── pso_optimizer.py        # PSO优化器
│   ├── nerve_model.py           # 神经模型
│   ├── data_importer.py         # 数据导入
│   ├── data_processor.py        # 数据处理
│   └── visualization.py         # 图表工厂
├── utils/                       # 工具层
│   ├── logger.py               # 日志
│   └── exceptions.py           # 异常定义
└── tests/                       # 测试
```

## 核心模块说明

### modules.pso_optimizer

自适应粒子群优化器，用于神经电刺激模型参数辨识。

**主要类**：

```python
class AdaptivePSO:
    """自适应PSO优化器"""
    def __init__(self, n_particles=100, n_iterations=50, dim=8,
                 bounds=None, c1=1.8, c2=1.5, w_max=0.9, w_min=0.4)
    def optimize(self, fitness_func, verbose=True, progress_callback=None)
```

```python
class NerveParameterOptimizer:
    """神经参数优化器"""
    def set_data(self, currents_A, responses, pulse_width_s)
    def optimize(self, n_particles=50, n_iterations=50, target_rmse=0.03)
    def compute_response_curve(self)
    def compute_threshold_current(self, target_p=0.5)
```

### modules.nerve_model

神经电刺激模型实现。

**主要类**：

```python
class RLCEquivalentCircuit:
    """RLC等效电路"""
    def simulate(self, current_A, pulse_width_s, dt=1e-6)

class ActivationFunction:
    """激活函数"""
    def compute(self, vc, dt)

class TibialNerveModel:
    """胫神经模型"""
    def predict(self, current_A, frequency_Hz, pulse_width_s)
```

### modules.data_importer

数据导入和解析。

**主要类**：

```python
class DataImporter:
    """数据导入器"""
    def import_data(self, file_path)
    def validate_data(self, df)

class NerveDataParser:
    """神经数据解析器"""
```

### ui.main_window

**主要组件**：
- 菜单栏

## 开发指南

### 添加新的图表类型

1. 在 `ui/chart_widgets.py` 中创建新类：

```python
class NewChartWidget(ChartWidget):
    def __init__(self, parent=None):
        super().__init__(parent)

    def plot(self, data, title=None):
        fig = Figure(figsize=(8, 6))
        ax = fig.add_subplot(111)
        # 绑定图表
        self.set_figure(fig)
```

2. 在 `main.py` 中注册图表：

```python
self.chart_widgets['new_chart'] = NewChartWidget()
self.chart_tabs.addTab(self.chart_widgets['new_chart'], "新图表")
```

## 测试指南

### 运行测试

```bash

pytest tests/test_pso_optimizer.py

pytest --cov=. --cov-report=html
```

```python
def test_new_feature():
    """测试新功能"""
    # Arrange
    optimizer = NerveParameterOptimizer()

    # Act
    result = optimizer.optimize(...)

    # Assert
    assert result.best_rmse < 0.1
```
