# 快速开始指南

本文档帮助您在5分钟内开始使用PSO数据分析工具。

## 前置准备

### 方式一：可执行程序（推荐）

1. 下载发布的可执行文件
2. 双击运行即可，无需安装Python

### 方式二：Python环境运行

```bash
# 1. 克隆或下载项目
cd PSO_UI

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行程序
python main.py
```

---

## 5分钟快速上手

### 第1分钟：启动程序

运行程序后，您将看到主界面：

```
┌─────────────────────────────────────────────────────────────┐
│  PSO数据分析工具                              [_][□][X]     │
├─────────────────────────────────────────────────────────────┤
│  菜单栏 | 工具栏                                           │
├────────────────┬────────────────────────────────────────────┤
│  左侧参数面板  │  右侧区域                                  │
│                │  ┌──────────────────────────────────────┐  │
│  [PSO参数]    │  │ [数据预览] [优化结果]                  │  │
│  [分析频率]    │  ├──────────────────────────────────────┤  │
│  [目标RMSE]   │  │                                      │  │
│                │  │  表格内容/图表内容                    │  │
│  [导入数据]    │  │                                      │  │
│  [运行优化]    │  └──────────────────────────────────────┘  │
│                │                                            │
└────────────────┴────────────────────────────────────────────┘
```

### 第2分钟：准备数据

准备您的EMG数据文件，格式如下：

**emg_processed.csv**：
```csv
I_peak(mA),Freq_10Hz,Freq_20Hz
5.0,0.05,0.08
6.0,0.12,0.18
7.0,0.25,0.35
8.0,0.42,0.55
9.0,0.58,0.72
10.0,0.71,0.85
```

### 第3分钟：导入数据

1. 点击工具栏的 **"导入"** 按钮
2. 或使用快捷键 **Ctrl+O**
3. 选择您的CSV文件
4. 数据将显示在"数据预览"Tab中

### 第4分钟：运行优化

1. 在左侧面板设置PSO参数（使用默认值即可）
2. 选择分析频率（如10Hz）
3. 点击 **"运行PSO优化"** 按钮
4. 等待优化完成（通常10-30秒）

### 第5分钟：查看结果

优化完成后：

1. **优化结果Tab**：查看辨识的参数和RMSE、R²等指标
2. **图表Tab**：查看7种可视化图表
   - 响应曲线
   - 收敛曲线
   - SD曲线 等

---

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+O | 导入数据 |
| Ctrl+S | 保存当前图表 |
| F5 | 刷新数据 |
| F9 | 运行PSO优化 |

---

## 默认参数配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 粒子数 | 50 | 搜索粒子数量 |
| 迭代次数 | 50 | 最大迭代代数 |
| 目标RMSE | 0.03 | 收敛精度要求 |
| 分析频率 | 10Hz | 刺激频率 |
